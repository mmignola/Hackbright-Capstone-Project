# Craft Tracker

## Description
Craft Tracker was my capstone project for the Hackbright Academy software engineering bootcamp. I've always been a big crafter and wanted a way to organize and track progress on all my projects. Craft Tracker allows users to create projects and label them with attributes (craft type, difficulty, etc) and filter through them by said attributes. Users can also create updates for their projects to track their progress and make notes. 

## Future Versions
I plan to continue working on Craft Tracker in the near future and am excited to develop features that I didn't have time to include while working on it at Hackbright. Features I plan to add include:

* An improved UI
* Adding cover photos to projects and updates
* Viewing and following other users' profiles
* Commenting on posts and updates

## Tech Stack
The tech stack included Python, HTML, SQLAlchemy, and CSS.
